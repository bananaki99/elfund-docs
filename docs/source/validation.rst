Δομή αρχείου επικύρωσης ΑΠΑ
===========================

Το σχήμα της δομής μπορείται να το κατεβάσετε από τις :doc:`γρήγορες λήψεις <download>`.  Το κάθε ΑΠΑ περιέχει τα ακόλουθα κλειδιά:

HEADER
    Περιέχει τα :doc:`βασικά στοιχεία <validation/header>` του αρχείου επικύρωσης. 

ERRORS
    Περιέχει τα :doc:`λάθη <validation/error>` του αρχείου επικύρωσης. 

WARNINGS
    Περιέχει τα :doc:`warnings <validation/warning>` του αρχείου επικύρωσης.
